# easy_music

这是一个使用Flutter框架开发的音乐应用，开发它的目的是学习Flutter开发相关知识。

# 已完成功能

## Getting Started

1. clone本项目
2. clone https://github.com/Binaryify/NeteaseCloudMusicApi
3. 启动网易云音乐node服务器
4. 编译并运行本项目
